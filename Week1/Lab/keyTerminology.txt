● Source code: Instructions written in a programming language that humans can understand.
● Machine code:  Code understood by machine
● Interpreter:  A program that reads and executes code line by line
● Compiler: Program that translate high level code to machine code
● 2GL, 3GL, 4GL: Generation of computer. 2GL(Assembly Language),3GL(Python, Java), 4GL(Javascript)
● Executable:A file containing machine code that can be directly executed by a computer 
● Expressions:Combinations of values and operators that produce a result in a programming language.
● Operators and Operands: Operators are symbols that perform operations, and operands are the values they operate on.
● Syntax Errors: Mistakes in the structure of the code 
● Logical Errors: Mistakes in the program's logic,
