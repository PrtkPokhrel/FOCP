● Module: A file containing Python code that you can use in other programs.

● The Python Standard Library: A collection of ready-to-use tools and functions that come with Python.

● Formal Parameters: Variables used in a function to represent values it needs.

● Actual Parameters (argument values): Real values you give to a function when you use it.

● Default and Keyword Arguments: Default values a function uses unless you say otherwise, and naming values when you use a function.

● Lambda Expression: A quick way to make simple, short functions without naming them.
